Powerful, light-weight, fast, free URL shortner

Built on Python, this is a simple yet powerful, light-weight, fast and free URL shortner.
It's currently under development, and is open-source on GitHub. Programmed by ashishagarwal2023 and some other fellow contributors.

This site is hosted on PythonAnywhere and the domain CutYourURL.tech is from TechDomains through the Github Student Dev Pack. All URLs from cutyoururl.tech are automatically redirected to here, meaning you will not have to paste the long URL thought.

Why and What's coming?

This URL shortner service is based on a free 1-year TechDomains's student plan's custom domain. Neither it earn any profit for me, nor I got a credit card or bank account to recieve payments to, there are literally no ads, no data is shared, etc. To support me, you can sponsor me on Patreon.

Please do not nuke me or my project, it is only 1-year validity domain, limited CPU and storage.

I'm actively working on this project. Currently I've added custom short links, link shortening, click tracking, authentication, recent shorts, QR code generating, reCAPTCHA and server-side logging. I'm actively spending like atleast 3 hours daily on this project to continue it. I'm planning to add more features like dashboard, API to short links, custom-length links, browser-extension, advanced options and lot more.
