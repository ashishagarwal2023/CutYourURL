Powerful, light-weight, fast, free URL shortner

Built on Python, this is a simple yet powerful, light-weight, fast and free URL shortner.
It's currently under development, and is open-source on GitHub. Programmed by ashishagarwal2023 and some other fellow contributors.

Why and What's coming?

This URL shortner service is based on a free 1-year NameCheap's student plan's custom domain. Neither it earn any profit for me, nor I got a credit card or bank account to recieve payments to, there are literally no ads, no data is shared, etc. To support me, just follow me on my GitHub.

(The Github Student Dev Pack is currently under processing, so till it doesn't gets done, I cannot have a domain for this.)

I'm actively working on this project. Currently I've added link shortening, click tracking, authentication, recent shorts and logging. I'm actively spending like atleast 3 hours daily on this project to continue it. I'm planning to add more features like custom short links, dashboard, API to short links, QR codes, custom-length links, QR codes, browser-extension, advanced options and lot more.
